# BombayDC Contact Us Page

A premium, high-performance, and fully responsive "Contact Us" page built for **BombayDC**. This project prioritizes aesthetics, smooth interactions, and modern web performance standards.

## 🚀 Live Demo Features

- **Premium Aesthetics**: Clean design using Museo Slab typography and a cohesive blue-accented color palette.
- **Interactive Map**: A custom map section featuring a "live" location indicator.
- **Custom Forms**: Polished enquiry and newsletter forms with interactive dropdowns and real-time focus states.
- **Dynamic Overlays**:
    - **Enquiry Modal**: A full-screen overlay for quick enquiry submissions.
    - **Mobile Menu**: A dedicated full-screen navigation menu for mobile users.
- **Scroll Memory & Locking**: The background automatically freezes when a menu or modal is open to ensure a smooth user experience.
- **Performance Optimized**: Includes font and image preloading, asynchronous image decoding, and layout shift (CLS) prevention.

## 🛠️ Built With

- **HTML5**: Semantic markup for better SEO and accessibility.
- **Vanilla CSS3**: Custom styles with a focus on modern layouts (Grid and Flexbox).
- **Vanilla JavaScript (ES6+)**: Custom logic for modals, menus, and form interactions—no heavy library dependencies.
- **Font Awesome**: High-quality vector icons for social media and UI elements.

## 📦 Project Structure

```text
├── assets/             # Project images (SVGs, PNGs)
├── fonts/              # Local custom font files (Museo Slab, Helvetica)
├── index.html          # Main entry point
├── script.js           # Core interaction logic
└── styles.css          # Global styling & animations
```

## ⚙️ Setup & Running Locally

1. **Clone or Download**: Download the project folder to your local machine.
2. **Launch**:
   - Simply double-click `index.html` to open it in your default web browser.
   - **Recommended**: Use a local dev server (like the "Live Server" extension for VS Code) for the best experience with real-time updates.

## 📈 Optimization

- **Preloading**: Critical assets (Fonts and Map Image) are pre-fetched to eliminate FOIT (Flash of Unstyled Text).
